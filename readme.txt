= Overview =
This is the plugin which helps you to upload the files for a specific customer orders.

= Features =

* The plugin is compatible with our <strong>Support Ticket System By Phoeniixx plugin</strong>
* Option to upload the specific pdf file for the specific customer orders.
* User can view the attachment in their my account.

[FREE VERSION DEMO LINK](http://orderattachmentfree.phoeniixxdemo.com/wp-admin) | [FREE VERSION DOCUMENTATION LINK](http://orderattachmentfree.phoeniixxdemo.com/wp-content/uploads/order-attachement.pdf)

== Installation ==

1. Install [WordPress](https://wordpress.org/)
2. Go to your WordPress admin panel, and then to Plugins. Click on "Add New" and then upload the zip file of the plugin using the "Upload Plugin" button you can find on top of the screen.
